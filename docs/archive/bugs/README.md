## üîÑ Current Workflow

### Bug Tracking (GitHub Issues)

1. **Report Bugs:** Create a new issue at [GitHub Issues](https://github.com/StiensWout/DLWait/issues)
2. **Categorize:** Use GitHub labels (`bug`, `critical`, `high-priority`, etc.)
3. **Track Progress:** Update issue status and link to PRs when fixed
4. **Close:** Mark issues as resolved when bugs are fixed

### Task Management (GitHub Projects)

1. **Source:** Tasks are defined in [TODO.md](./TODO.md)
2. **Sync to GitHub:** Use the [GitHub Projects Sync Guide](./GITHUB_PROJECTS_SYNC.md) to create issues for tasks
3. **Organize:** Add tasks to your GitHub Project board
4. **Track:** Use GitHub Projects columns (To Do, In Progress, Done, etc.)
